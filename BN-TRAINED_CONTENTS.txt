MLE ESTIMATION PROCESS:
1. For each node in topological order
2. Get parent state combinations
3. Count frequency of each (parent combo, child state) pair
4. Probability = Count / Total occurrences of parent combo
5. Apply Laplace smoothing: (Count + 1) / (Total + num_states)

  - Cannot validate generalization
  - High variance in parameter estimates
  - CPD7 (Active_TB) only 3/36 parent combos observed
